	Lead-to-Clio integrates your Wordpress Blog with your Clio Account - automatically creating tasks and contacts for new leads.
	
	== Description ==
	
	Lead-to-Clio provides a contact form of first name, last name, phone number and email that integrates with the Clio Online Legal Practice Management Software. If the user has authorized Lead-to-Clio to access the user's Clio account, when a prospective client submits their contact info on the form a new Contact entry is created in Clio, a new task entry is created in Clio and a notification email is sent to the Clio users.
	A few notes about the sections above:
	
	== Installation ==
	
	1. Upload `Lead-to-Clio` folder to the `/wp-content/plugins/` directory
	2. Activate the plugin through the 'Plugins' menu in WordPress
	3. On the 'Lead-to-Clio' Settings page (found from your Wordpress dashboard) - click on the "Authorize Clio" link.
